# Levex Computerized Maintenance Management System
http://levexmms.herokuapp.com

![Levex MMS](./public/imgs/Screenshot%202022-10-09%20at%2014-09-14%20Levex%20MMS.png)

## How It's Made:
**Tech Used:** Node.js, MongoDB, Express, JavaScript, CSS, EJS

Levex is a CMMS used to create, track, and complete work requests. The site was designed based on my exprience using MMS at previous jobs. The logic is fairly straightforward and is basically a CRUD app. 

## Optimizations
